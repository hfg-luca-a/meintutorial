Google Fonts

font-style

font-style: Spezifiziert den font style (normal, italic, oblique). Default Wert ist "normal".Test (fontstyle01.html)

font-variant

font-variant: Spezifiziert die font variant (small-caps, initial, inherit). Default Wert ist "normal". Test (fontvariant01.html)

font-weight

font-weight: Spezifiziert das font weight (normal, bold, Zahlenwert zw. 100 und 900). Default Wert ist "normal". Test(fontweight01.html)

text decoration

text-decoration: Spezifiziert verschiedenen Textauszeichnungen wie overline, line-through, underline, undeline overline. Default Wert ist "normal". Test (textdecoration01.html)

letter spacing

letter-spacing: Spezifiziert den Abstand zwischen den Buchstaben (Sperrschrift) Test (letterspacing01.html)

inherit keyword

Das inherit keyword spezifiziert, dass eine Eigenschaft vom Elternelement vererbt wird.
Das inherit keyword kann für jede CSS Eigenschaft und jedes HTML Element benutzt werden. Test (inherit01.html)
